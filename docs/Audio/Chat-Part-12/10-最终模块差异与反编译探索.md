2. **喇叭选择**：通过 `AudioManager.setParameters("speaker_force=1")` 设置 HAL 参数强制路由到指定喇叭
   - `speaker_force=` — 0号喇叭
   - `speaker_1_force=` — 1号喇叭
   - ...以此类推，最多6个
3. **音量**：`setStreamVolume(STREAM_MUSIC, maxVolume, 0)` 拉到最大
4. **Smart PA 校准**：也是 `MediaPlayer` 播放 + `getParameters("emsmartpacalirange")` / `getParameters("emsmartpacaliresult")` 查询校准结果

**这对我们排查的意义：**

工程模式走的是 `MediaPlayer → AudioFlinger → AudioHAL → AGM → DSP → PA` 完整 Android 音频栈，在 recovery 里无法直接用（没有 AudioFlinger/AudioServer）。
